⚡ How to Run

Requires Java 17+ and Maven 3.8+

cd plagiarism-checker
mvn clean package -DskipTests
mvn spring-boot:run
# open http://localhost:8080
🔬 7 Algorithms
  • Rabin-Karp — rolling hash k-gram overlap
  • KMP — exact phrase detection
  • Winnowing — MOSS fingerprinting
  • Levenshtein — sentence edit distance
  • Cosine TF — vector space model
  • Jaccard — set-based comparison
  • Synonym Map — paraphrase detection
📦 Test Datasets
  • PAN Corpus pan.webis.de — 50k+ labelled pairs
  • Clough & Stevenson — 5 plagiarism types
  • MRPC (Microsoft) — 5801 paraphrase pairs
  • METER Corpus — journalism rewrites
  • Project Gutenberg — public domain books
🔌 API Reference

Core Analysis:

POST /api/analyze/text — Compare two texts

POST /api/analyze/files — Compare uploaded files

POST /api/v2/analyze — Analyze texts with history tracking

POST /api/v2/analyze/files — Analyze files with history tracking

Batch Operations:

POST /api/v2/batch — Batch N×N comparison matrix

Reports:

POST /api/v2/report/html — Export HTML report

History:

GET /api/v2/history — Get analysis history

DELETE /api/v2/history/{id} — Delete specific history entry

DELETE /api/v2/history — Clear all history

GET /api/v2/history/summary — Get history summary

Synonyms:

GET /api/v2/synonyms?word=fast — Lookup synonyms for a word

GET /api/v2/synonyms/check?a=fast&b=quick — Check if words are synonyms

Internet Scan:

POST /api/v2/internet-scan — Scan internet for plagiarism matches

Health:

GET /api/health — Service health check

🛡️ Security Features
  • Rate Limiting — 60 req/min per IP
  • Input Validation — All inputs sanitized
  • CORS — Configurable origins
  • File Size Limits — 10MB max upload
  • Timeout Protection — Web fetch timeouts
